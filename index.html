Kafka Installation Guide for On-Premise Systems

This guide covers how to install Apache Kafka on an on-premise system.

1. Prerequisites

  • Linux-based OS (RHEL, Ubuntu, etc.)
  • Java 11 or later
  • Zookeeper

2. Install Java

sudo yum install java-11-openjdk -y

3. Download and Extract Kafka

wget https://downloads.apache.org/kafka/3.5.0/kafka_2.13-3.5.0.tgz
tar -xvzf kafka_2.13-3.5.0.tgz
cd kafka_2.13-3.5.0

4. Start Zookeeper

bin/zookeeper-server-start.sh config/zookeeper.properties

5. Start Kafka Server

bin/kafka-server-start.sh config/server.properties

6. Create a Kafka Topic

bin/kafka-topics.sh --create --topic test-topic --bootstrap-server localhost:9092

7. Verify Kafka Installation

bin/kafka-topics.sh --list --bootstrap-server localhost:9092
